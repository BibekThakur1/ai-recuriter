# AI Recruiter

A simple AI recruiting app where recruiters can post jobs, share interview links, and review candidate answers with AI scoring.

Live: https://ai-recuriter-xi.vercel.app

## Features

- Recruiter and candidate login
- Job posting
- AI interview questions
- Candidate interview link
- Candidate scoring and review
- Hiring pipeline

## Tech Used

Next.js, Tailwind CSS, Clerk, Supabase, OpenAI, Vercel

## Run Locally

```bash
npm install
npm run dev
```

Create `.env.local` and add Clerk, Supabase, and OpenAI keys.

## Note

This is a portfolio project built to show a complete hiring flow from job posting to candidate review.
